# 05. Форматированные подписи с табуляциями

## Суть

Некоторые TextMesh в игре содержат **табуляции** (`\t`, код 0x09) для выравнивания колонок — прежде всего в настройках управления. Структура: `Действие\t\tКлавиша`, где несколько табуляций задают отступ между колонками. Наивный перевод (замена всей строки целиком через онлайн-сервис) разрушает формат.

## Симптом

При переводе строки `Walk\tForward\tStop` через Google Translate:
- Переводчик воспринимает `\t` как разделитель или игнорирует его.
- Результат может содержать literal `/t/t/t` или потерянные колонки.
- Подписи в настройках «разъезжаются» — текст больше не совпадает с кнопками.

## Решение: токенизация по разделителям

Перевод выполнять по сегментам, разделённым управляющими символами, с сохранением оригинальных разделителей при сборке:

```
Оригинал:     Walk\tForward\tStop
                ↓
Токенизация:  [Walk] [\t] [Forward] [\t] [Stop]
                ↓
Перевод:      [Идти] [\t] [Вперёд] [\t] [Стоп]
                ↓
Сборка:       Идти\tВперёд\tСтоп
```

Разделители для токенизации: `\t` `\n` `\r` `\f` `\v` (все стандартные escape из C-format).

## Реализация

```csharp
// Разбиение с сохранением разделителей.
// Последовательные разделители НЕ схлопываются — "Walk\t\tForward"
// даёт [Walk] [\t] [\t] [Forward], что сохраняет ширину колонки.
List<Token> Tokenize(string text) {
    var tokens = new List<Token>();
    var current = new StringBuilder();
    foreach (char c in text) {
        if (IsSeparator(c)) {
            if (current.Length > 0) {
                tokens.Add(Text(current.ToString()));
                current.Clear();
            }
            tokens.Add(Delimiter(c));
        } else {
            current.Append(c);
        }
    }
    if (current.Length > 0) tokens.Add(Text(current.ToString()));
    return tokens;
}
```

Каждый текстовый сегмент переводится независимо (через кеш или онлайн-сервис). Уже переведённые сегменты (содержащие кириллицу) пропускаются. Технические сегменты (коды клавиш, числа) фильтруются (см. [06-keycode-strings.md](06-keycode-strings.md)).
